# Neural Networks From Scratch

A complete implementation of neural networks using only **NumPy**, built while studying *Neural Networks from Scratch* (NNFS).

The goal of this repository is to understand every mathematical concept behind deep learning instead of relying on high-level frameworks such as TensorFlow or PyTorch.

---

## About

This project focuses on implementing every component of a neural network from first principles, including:

- Forward propagation
- Backpropagation
- Gradient descent
- Activation functions
- Loss functions
- Optimizers
- Regularization
- Training on real datasets

Every algorithm is implemented manually using NumPy to gain an in-depth understanding of how modern deep learning libraries work internally.

---

## Features

- Dense (Fully Connected) Layers
- ReLU Activation
- Softmax Activation
- Categorical Cross Entropy Loss
- Combined Softmax + Cross Entropy Backpropagation
- L1 & L2 Regularization
- SGD Optimizer
- SGD with Momentum
- AdaGrad
- RMSProp
- Adam Optimizer
- Learning Rate Decay
- Spiral Dataset Classification
